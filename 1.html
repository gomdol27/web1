HTML

Hypertext Markup Language (HTML) is the stnadard markup language for creating web pages and web applications. Web browsers receive HTML documents from a web server of from loval storage and render them into multimedia web pages. HTML describes the structure of a web page semantically and originally included cues for the appearance of the document.

HTML elements are the buliding blocks of HTML pages. With HTML constructs, images and other objects, such as interactive forms, mat be embeded into the rendered page. It provides a means to create structured documents by denoting structual semantics for text such as headings, paragraphs, lists, links, quotes, and other items. HTML elements are delineated by tags, written using angle brackets.
